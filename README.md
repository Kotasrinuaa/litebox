## 📘 What is litebox?

litebox is a simple tool designed to protect your computer by running programs in a safe way. It acts like a small operating system within your computer. This helps keep your main system safe from changes or threats that might come from the programs you use.

This software works with different parts of your computer system to provide strong protection. It runs both the main computer section (called the kernel) and normal programs (user mode) safely, so your computer stays secure.
